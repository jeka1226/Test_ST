Результат выполненния тестового задания.

Программа протестирова на Windows 10 и Debian 11.
Для запуска на Windows достаточно ванильного python 3.8
Для запуска на Linux требуется дополнительно установить gnome-terminal

Если у вас заняты порты 12345 и 12346, то их необходимо освободить, либо изменить настройки программы.

Запуск сервера в терминале:
python StartServer.py

Запуск клиента в терминале:
python StartClient.py

После запуска клиента stdin будет всё время ожидать от вас ввода команды. 
Кроме того, откроется второе терминальное окно, в котором будут выводиться результаты обработки запросов сервером.

Введите help и нажмите enter, чтобы посмотреть справку по всем командам.

Если коротко, то запросы имеют следующую форму.
Отправка задачи на сервер (перевернуть слово):
	task --reverse your text

Отправка задачи на сервер в пакетном режиме (перевернуть слово):
	task --reverse -b your text
Чтобы выйти из пакетного режима нажмите CTRL+C.

Проверка статуса задачи (identifier - число)
	status identifier

Проверка результата исполнения задачи (identifier - число)
	result identifier

Посмотреть все идентификаторы:
	identifiers

Чтобы выйти из клиента или из сервера нажмите CTRL+C.
